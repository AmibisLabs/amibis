isValid

boolean isValid(Service service,
                java.lang.String variableName,
                java.lang.String newValue)
This method is called when a new value is requested on a variable. This method must check that this new value is a valid value. A listener throwing an exception in the #isValid call will be considered as agreeing the change/returning true.

Parameters:
service - the service owning the variable
variableName - the name of the variable on which the modification requested is happening
newValue - the new requested value
Returns:
true if the new value is accepted, false if rejected.
