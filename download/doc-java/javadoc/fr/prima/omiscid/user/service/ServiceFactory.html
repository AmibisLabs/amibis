createServiceRepository

ServiceRepository createServiceRepository(Service service)
Creates a new ServiceRepository based on a already created service. A service repository is used to mainting automatically a list of running services and can notify of services apparition and disapparition. If you are only willing to look for services at a given instant, using Service.findService(fr.prima.omiscid.user.service.ServiceFilter) methods is simpler. Connections that may be necessary for the service repository will be done in the name of the provided service.

Parameters:
service - a Service in the name of which the connection necessary for the service repository will be done
Returns:
an operational ServiceRepository

createServiceRepository

ServiceRepository createServiceRepository()
Creates a new ServiceRepository. A service repository is used to mainting automatically a list of running services and can notify of services apparition and disapparition. If you are only willing to look for services at a given instant, using Service.findService(fr.prima.omiscid.user.service.ServiceFilter) methods is simpler. A new Service instance will be hiddenly created. If you already have a Service instance, you should probably prefer createServiceRepository(Service). Connections that may be necessary for the service repository will be done in the name of the hiddenly created service.

Returns:
an operational ServiceRepository
