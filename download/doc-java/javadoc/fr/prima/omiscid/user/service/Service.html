                     byte[] msg,
                     int pid,
                     boolean unreliableButFastSend)
                     throws UnknownConnector
Sends a message to a particular client. This client is identified by its Peer id (pid). This method is usually used to answer a request coming from another service that has requested a connexion with us. We know this service from its pid inside its request message. We do not have a bipServiceProxy for it because we have not found this service to initiate the connexion.

Parameters:
connectorName - the name of the connector that will send the message
msg - the message to send
pid - peer id : the identification of the client that must receive the message
unreliableButFastSend - not implemented yet
Throws:
UnknownConnector - thrown if the service has not declared this connector
