connected

void connected(Service service,
               java.lang.String localConnectorName,
               int peerId)
Called when a remote service connects to the local connector. The peerId is a unique pair (service + remote connector). The corresponding service filter can be found using a PeerIdIs filter. The connector name can be found using the findConnector method on the ServiceProxy

Parameters:
service - the service receiving the message
localConnectorName - the name of the connector handling the broken link
peerId - the peerId (a unique Id corresponding to a connector and a remote service)
